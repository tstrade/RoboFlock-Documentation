Arduino Nano
^^^^^^^^^^^^

**Datasheet:** :raw-html:`<br />`
`Arduino Nano ATmega328 Microcontroller User Manual <nano_>`_


**Features:** :raw-html:`<br />`
16MHz Clock Frequency :raw-html:`<br />`
Digital, Analog, PWM, UART, I2C, and SPI Capabilities :raw-html:`<br />`
32kB Flash Memory :raw-html:`<br />`
2kB Internal SRAM :raw-html:`<br />`
1kB EEPROM :raw-html:`<br />`
Ideal for Constrained Spaces


**Connection:** :raw-html:`<br />`
Mini-B USB to USB Type-A 


**Interfacing:** :raw-html:`<br />`
`Arduino IDE <ide_>`_ 


**Dimensions (with headers):** :raw-html:`<br />`
43.2mm x 18mm x 18.5mm


**Use-Case:** :raw-html:`<br />`
Controls and processes data for handheld beacon.



Arduino Uno
^^^^^^^^^^^

**Datasheet:** :raw-html:`<br />`
`Arduino UNO R3 ATmega328P Microcontroller User Manual <uno_>`_


**Features:** :raw-html:`<br />`
16MHz Clock Frequency :raw-html:`<br />`
Digital, Analog, PWM, UART, I2C, and SPI Capabilities :raw-html:`<br />`
32kB Flash Memory :raw-html:`<br />`
2kB Internal SRAM :raw-html:`<br />`
1kB EEPROM :raw-html:`<br />`
Direct Shield Compatibility


**Connection:** :raw-html:`<br />`
USB Type-B to USB Type-A


**Interfacing:** :raw-html:`<br />`
`Arduino IDE <ide_>`_ 


**Dimensions:** :raw-html:`<br />`
68.6mm x 53.4mm x 12.5mm


**Use-Case:** :raw-html:`<br />`
Controls and processes data for all three :doc:`ultrasonic sensors <hc_sr04>`.
