Ultrasonic Distance Sensor
==========================

.. include:: ../_templates/constants.rst
    

**Datasheet:** :raw-html:`<br />`
`Ultrasonic Ranging Module HC-SR04 <datasheet_>`_


**Features:** :raw-html:`<br />`
2cm to 400cm Measurement Range :raw-html:`<br />`
3mm Ranging Accuracy :raw-html:`<br />`
40Hz Working Frequency


**Connection:** :raw-html:`<br />`
*VCC* to 5V Supply :raw-html:`<br />`
*TRIG* to Arduino Digital Pin :raw-html:`<br />`
*ECHO* to Arduino Digital Pin :raw-html:`<br />`
*GND* to 0V Ground


**Interfacing:** :raw-html:`<br />`
`Arduino Servo Library <servo_>`_ 


**Dimensions:** :raw-html:`<br />`
45mm x 20mm x 15mm


**Use-Case:** :raw-html:`<br />`
Detects close-range obstacles for the front-facing portion of RoboFlock. Acts as a failsafe for objects not detected by the LiDAR module.



.. _datasheet: https://cdn.sparkfun.com/datasheets/Sensors/Proximity/HCSR04.pdf

.. _servo: https://github.com/arduino-libraries/Servo
